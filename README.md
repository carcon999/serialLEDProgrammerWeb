serialLEDProgrammer For Web
===================

### 概要

フルカラーシリアルLED（WS2812系）を制御するLED演出コマンドをブラウザ上で作成できます。作成したLED演出をArduino用シールドや、開発したフルカラーLEDコントローラを介して演出することが可能になります。
